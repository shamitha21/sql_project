# Data Analyst Job Market Analysis using SQL

## Introduction

This project explores the data analyst job market using SQL. The analysis focuses on salary trends, in-demand skills, and the most valuable skills for data analysts based on real-world job posting data.

The entire project was built using SQL queries, helping me strengthen both my SQL fundamentals and advanced querying skills.

## Tools Used

* **SQL** – Data analysis and querying
* **PostgreSQL** – Database management system
* **Visual Studio Code** – Writing and running SQL queries
* **Git & GitHub** – Version control and project sharing

## Key Questions Answered

* What are the highest-paying data analyst jobs?
* What skills are required for top-paying jobs?
* Which skills are most in demand?
* Which skills are associated with higher salaries?
* What are the best skills to learn for data analysts?

## Project Highlights

### Top-Paying Data Analyst Jobs

Analyzed remote data analyst roles with the highest salaries and identified companies offering top compensation packages.

### Skills for High-Paying Roles

Found that skills like **SQL, Tableau, Excel, and Python** frequently appear in top-paying job postings.

### Most In-Demand Skills

SQL emerged as the most requested skill in data analyst job postings, followed by Excel, Tableau, and Power BI.

### Highest Paying Skills

Explored which technical skills are linked to higher average salaries in the data analytics field.

### Optimal Skills to Learn

Combined salary and demand analysis to identify the most valuable skills for aspiring data analysts.

## What I Learned

This project helped me improve my understanding of:

* SQL basics and advanced queries
* JOINs and CTEs
* Aggregate functions like `COUNT()` and `AVG()`
* Filtering, grouping, and sorting data
* Real-world data analysis using SQL

## Conclusion

This project strengthened my SQL skills by applying both basic and advanced queries to real-world datasets. It also provided useful insights into the data analyst job market, including salary trends and the most valuable skills to learn for career growth in data analytics.
